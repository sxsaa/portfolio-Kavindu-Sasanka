Latest Projects

SafeLABS

Comprehensive Laboratory Management System with user access authentication system for a chemical laboratory. SafeLABS contain face recognition based user authentication with the integration of spoofing detection and object detection to verify safety compliance. SafeLABS system also maintains user, resource & other records and features automated attendance recording. The users are able to view lab occupancy details, announcements, and resource details through a web application.


Tech Stack : React, Redux, Tailwind CSS, Microsoft .NET, C#, MSSQL Server, OpenCV, Python, YOLO

CuddleCam

A portable and automated baby monitoring system which uses computer vision techniques to notify the parents through a mobile app whenever a risky situation occurs.


Tech Stack : Python, OpenCV, TensorFlow, Neural Networks, Computer Vision, MediaPipe, Flask, Flutter

MotionMetrics

Sports Biomechanics Pose Analysis System for multi-angle pose evaluation of athletes using a synchronized multi-camera setup. The system captures key sports movements such as jumps and gaze direction through marker-based joint tracking. It performs 3D reconstruction of body poses using geometric triangulation and calibration techniques. The system supports athlete-specific session analysis, pose visualization, and basic motion analytics through an integrated desktop interface.

Tech Stack : Python, OpenCV, TensorFlow, Neural Networks, Computer Vision, MATLAB
